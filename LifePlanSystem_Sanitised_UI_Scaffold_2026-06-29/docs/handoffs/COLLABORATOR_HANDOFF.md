# Collaborator Handoff

This is a clean handoff for helping build the LifePlanSystem UI.

## Build goal

Create a UI/pass-through layer for a Markdown/GitHub-backed AI personal operating system.

## Keep in mind

- Repository remains source-of-truth.
- UI should not hide provenance.
- AI should propose changes, not silently commit.
- Historical records should be preserved.
- Sensitive files should be protected.
- Start with a small MVP.

## Best first prototype

- Dashboard.
- File browser.
- Chat with selected files.
- Approval queue.
- Staged proposal preview.
