# Sanitization Manifest

This export was created as a curated public-safe archive.

## Removed / not included

- Private memory files
- Sensitive context files
- Personal profile files
- Personal career and health records
- Personal admin todos
- Chat checkpoint files likely to contain personal context
- Any files containing direct personal names, relationship details, legal details, health details, employment specifics, or other identifying content

## Included

- Generic memory architecture
- Memory metadata standard
- Generic rule wording
- Memory classification rule
- Repository write fallback checklist
- Public-safe support-router template
- Public-safe source-of-truth placeholder

## Limitation

The container could not clone the live GitHub repository because DNS access failed. This archive is therefore a curated sanitized export assembled from the current known safe project files and not a byte-for-byte sanitized clone of the full repository.
