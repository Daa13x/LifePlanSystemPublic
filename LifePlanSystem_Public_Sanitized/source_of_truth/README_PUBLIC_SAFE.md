# Source of Truth — Public-Safe Placeholder

The private repository may contain source-of-truth files with personal information.

This sanitized export intentionally excludes those files.

For public review, the important concept is the structure:

- promoted facts should be reviewed before becoming canonical;
- memories should begin in an inbox or holding area;
- sensitive records should remain isolated;
- corrections should be preserved rather than silently overwritten;
- personal data should not be included in public exports.
