# AI Repo Review Prompt

You are reviewing a Markdown/Git-backed personal operating system project.

Review for:

- architecture clarity;
- source-of-truth discipline;
- memory lifecycle;
- review and confirmation safety;
- UI feasibility;
- duplication;
- missing templates;
- privacy risks;
- anti-sycophancy;
- maintainability;
- smallest useful next steps.

Output:

1. What the system is trying to do.
2. What is already strong.
3. What is unclear.
4. What could break.
5. What to build first.
6. What not to build yet.
7. Suggested file changes.
8. Questions for the project owner.
