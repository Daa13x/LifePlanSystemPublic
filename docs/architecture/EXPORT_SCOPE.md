# Export Scope

This repository is a public-safe system architecture export.

## Goal

Preserve the reusable LifePlanSystem design so another person or AI can understand the system and help build the UI.

## Include

- generic rules;
- architecture notes;
- UI specifications;
- workflow diagrams;
- empty templates;
- placeholder memory files;
- approval flow examples;
- source review methods;
- model routing concepts;
- recovery and workload patterns written generically.

## Exclude

- real user memory;
- real user source-of-truth entries;
- real chat logs;
- real personal examples;
- raw uploaded documents;
- screenshots with identifying content.

## Method

Where a file in the original project contains reusable system design mixed with user-specific content, create a clean equivalent here using placeholder examples.

## Rule

This public repo should explain how the system works, not reveal the original user's life data.
